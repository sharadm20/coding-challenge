# Coding Challenge Repository

A comprehensive collection of LeetCode and interview coding challenges designed for practice and skill development.

## 📋 Overview

This repository contains solutions and implementations for various coding challenges, including LeetCode problems and common interview questions. It serves as a learning resource for improving problem-solving skills across multiple programming paradigms.

## 🛠️ Technology Stack

This project is built with multiple programming languages to provide diverse perspectives and solutions:

| Language | Percentage | Purpose |
|----------|-----------|---------|
| **Java** | 79.6% | Primary implementation language for most solutions |
| **Rust** | 13.1% | Memory-safe implementations and performance-critical solutions |
| **Python** | 6.5% | Quick prototyping and algorithmic demonstrations |
| **JavaScript** | 0.8% | Web-based utilities and browser-compatible solutions |

## 🚀 Getting Started

### Prerequisites

- Java 8 or higher (for Java solutions)
- Python 3.7+ (for Python solutions)
- Rust 1.56+ (for Rust solutions)
- Node.js 14+ (for JavaScript solutions)

### Clone the Repository

```bash
git clone https://github.com/sharadm20/coding-challenge.git
cd coding-challenge
```

## 💡 How to Use

1. Navigate to the language-specific directory
2. Review the problem statement in comments or documentation
3. Study the solution implementation
4. Try to solve the problem independently first
5. Compare your approach with the provided solution
